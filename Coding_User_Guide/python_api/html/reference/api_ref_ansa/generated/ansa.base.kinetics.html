Module ansa.base.kinetics#

ansa.base.kinetics.FlexToolsExport(type: str, flex_body: Entity, kin_result: Entity, out_file_name: str, disp: bool = False, velo: bool = False, acce: bool = False, stress: bool = False, strain: bool = False, geometry: bool = False, trans: bool = False, rot: bool = False) → int#
The function exports a Flex Body’s results to various formats.
Parameters:
typestr
The type of the exported results: nodal, modal or nastran_lc
flex_bodyEntity
The flex body entity whose results are exported.
kin_resultEntity
The KinResults entity.
out_file_namestr
The output filename. The appropriate file extension, based on the type of the exported results, must be included in filename. For type=’nodal’ the file extensions ‘.op2’ or ‘.h5’ should be used. For type=’modal’ the extension ‘.pch’ should be used. For type=’nastran_lc’ all the extensions compatible to Nastran can be used.
dispbool, optional
Exports displacements. Valid for types ‘nodal’ and ‘modal’.
velobool, optional
Exports velocities. Valid for types ‘nodal’ and ‘modal’.
accebool, optional
Exports accelerations. Valid for types ‘nodal’ and ‘modal’.
stressbool, optional
Exports stresses. Valid for type ‘nodal’.
strainbool, optional
Exports strains. Valid for type ‘nodal’.
geometrybool, optional
Exports the geometry. Valid for type ‘nodal’.
transbool, optional
Exports translational component of force. Valid for type ‘nastran_lc’.
rotbool, optional
Exports rotational component of force. Valid for type ‘nastran_lc’.
Returns:
int
Returns 0 on success, 1 on failure.

Notes

At least one optional argument must be set to True. The argument must comply with the result type. A Kin Results entity should exist within the database and the flex body should participate in this results entity.

Examples

import ansa
from ansa import base
from ansa import constants
from ansa import kinetics

def main():
    flex_body = base.GetEntity(constants.NASTRAN, "KIN_BODY", 2)
    kin_res = base.GetEntity(constants.NASTRAN, "KIN_RESULTS", 1)
    filepath = "/users/demo/results/res_for_flex.op2"
    kinetics.FlexToolsExport(
        "nodal",
        flex_body,
        kin_res,
        filepath,
        disp=True,
        velo=True,
        acce=True,
        stress=True,
    )
