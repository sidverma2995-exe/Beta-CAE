ansa.base.checks.penetration.Intersections() → Check#
This function creates an appropriate Check object, that when executed, will perform
the same checks that are performed by the relevant ANSA Check.

Attributes and default values:
- check_same_pids = 1

Fix Attributes and default values:
- method = 1
- allow_edges_swap = 1
- expand_zones = 5
- max_movement = None

Returns:
Check
Returns the created ‘base.Check’ object.

See also

ansa.base.Check

Notes

Accepted values:
method: 0, 1, 2, 3 where,
0: opens ‘Fix intersections’ window
1: “Move Away”
2: “Collapse”
3: “Intersect Entities”

allow_edges_swap: 0, 1, where,
0: False
1: True

Examples

import ansa
from ansa import base

def main():
    obj = base.checks.penetration.Intersections()
    results = obj.execute()

    if results:
        header = results[0]
        try:
            header.try_fix()
        except RuntimeException as e:
            print("something went wrong")
