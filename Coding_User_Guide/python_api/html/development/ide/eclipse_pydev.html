Integrating with Eclipse IDE#

Introduction#

Eclipse is an Integrated Development Environment (IDE). It contains a base workspace and an extensible plug-in system for customizing the environment. In order for Eclipse to support python, a third party plug-in is needed, called PyDev. PyDev supports autocompletion, documentation, code refactoring, graphical debugging, code analysis and many other features. Eclipse and PyDev can be installed following the instructions given in their respective sites.

ANSA and META autocompletion#

A package with the necessary files that will enable autocompletion and documentation for BETA APIs is available here.

Installation Instructions#

  1. The first step is to download and install Eclipse. There are various packages available for download. The “Eclipse IDE for C/C++ Developers” package is recommended.

  2. In order for Eclipse to support the Python programming language, the PyDev plugin needs to be installed.

  3. A local system installation of Python 3.11 is also required.

Working in Eclipse#

In order to enable the Python programming environment the Pydev Perspective needs to be enabled as shown in the image below.

A new PyDev project needs to be started in order to specify common settings and preferences for all the scripts that will be developed for a scripting project. PyDev projects are created by clicking on File > New > PyDev Project. Python 3 needs to be selected for Grammar Version. This will allow Eclipse to check the syntax errors based on the Python 3 structure.

After the project creation, the path of the files provided by BETA for autocompletion needs to be typed in the project’s Properties. The path can be set by clicking on: Properties > PyDev - PYTHONPATH > External Libraries. The path of the directory where the pypredef files exist should be selected. Either pydev_ansa or pydev_meta, for ANSA or META autocomplete respectively. Then, by pressing “Force restore internal info” and “Apply” the autocompletion feature will be immediately available, otherwise the changes will be available after restarting Eclipse.

After setting the paths, autocompletion and documentation is available in any python module within the project.
