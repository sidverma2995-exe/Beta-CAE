Integrating with Sublime Text#

Introduction#

Sublime Text is as described a ‘sophisticated’ source code editor. It is a complete text editor for code, markup and prose, providing a fully customizable user interface, extraordinary features such as Multiple Selections and Distraction Free Mode, as well as having amazing performance. Sublime Text can be downloaded and installed following the instructions provided in its website.

ANSA and META autocompletion#

A package with the necessary files that will enable autocompletion and documentation for BETA APIs is available here.

Installation Instructions#

  1. The first step is to download and install Sublime Text. Sublime Text 3 is recommended, as it contains all the latest features.

  2. In order for Sublime Text to provide autocompletion for Python, it is recommended to instal a package called ‘Anaconda’.

Working in Sublime Text#

After installing and opening Sublime Text, the Package Control needs to be installed by clicking on Tools > Install Package Control.

Then, in order for ‘Anaconda’ to be installed the next steps should be followed: Preferences > Package Control > Install Package > Anaconda

After the Anaconda installation, the path of the files provided by BETA needs to be added in the Anaconda configuration file, by clicking on: Preferences > Package Settings > Anaconda > Settings - User. The following text has to be added, entering the path of the beta files. The pydev_ansa folder for ANSA autocompletion and the pydev_meta for META autocompletion:

{
    "extra_paths":
    [
        "/Users/user/Documents/sublimetext3_ansa"
    ]
}

After setting the path and restarting Sublime Text 3, the autocompletion and documentation feature is available during the editing of any python file.
