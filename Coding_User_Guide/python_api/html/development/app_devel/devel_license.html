BETA Developer License#

Introduction#

BETA CAE Systems has introduced a new license feature in ANSA and META, called BETA Developer License. This license is intended for developing script applications using the ANSA and META Python API, without having to acquire a full license. Limitations and reduced functionality come along with the Developer License.

ANSA#

Launching ANSA#

In order to launch ANSA in Developer License mode, use the “–developer-license” option:

ansa64.sh --developer-license

ansa64.bat --developer-license

The ANSA Launcher won’t appear. Instead, ANSA will open directly and its main window title will contain the label “Developer License”.

Limitations#

  • When the model contains more that 1000 elements or more than 50 faces:

    • Output of neutral deck files (File > Output) is disabled. API output functions like base.OutputNastran always return failure. However, parsing of arguments is performed, so that an error free script can be written.

    • Output of CAD files (File > Output CAD) is disabled.

    • Output from Database Browser’s right click Output option is disabled.

    • API functions return modified double values. The modification is around 1.5% for values less than 1 and goes down to 1e-06% for values greater than 10e14. Zero, whole doubles, nan, inf and -inf are returned unmodified. The modified functions are the following:

      • ansa.base.CalcElementMass()

      • ansa.base.CalcShellArea()

      • ansa.base.CalcSolidVolume()

      • ansa.base.ElementQuality()

      • ansa.base.Entity.get_entity_values()

      • ansa.base.Entity.position()

      • ansa.base.GetEntityCardValues()

      • ansa.base.GetLoadCurveData()

      • ansa.base.GetShellEffectiveEngineeringProperties()

      • ansa.base.Sgraph()

  • ANSA Databases are saved encrypted. When saving a database, a respective message is shown in ANSA Info. A database saved in Developer License mode can be opened only by ANSA running under this license. If you try to open it with normal ANSA, a denial message will appear in ANSA Info. Saving as previous version (File > Save as v20) is disabled. The API function base.SaveAs(filename=’foo.ansa’, version=’v20’|’previous’) returns failure.

  • Epilysis is blocked. In ANSA, the button NASTRAN > EPILYSIS > SOLVE is removed. API functions base.Epilysis and base.OutputEpilysis return failure.

META#

Launching META#

In order to launch META in Developer License mode, use the “–developer-license” option:

meta_post64.sh --developer-license

meta_post64.bat --developer-license

The META Launcher won’t appear. Instead, META will open directly and its main window title will contain the label “Developer License”

Limitations#

  • For 3D models with more than 1000 elements and curves with more than 50 points:

    • All 3D result values and curve point values read in META will be modified. The outcome of NVH calculators, the Linear Combination tool, the Random Response tool and the Section Forces tool will also be returned modified.

    • It won’t be possible to export to formats other than META databases and project files.

    • All script functions that return values related to geometry will return modified values.

  • A database saved in Developer License mode can only be opened by META running under this license. If you try to open it with normal META, a denial message will appear in the Info window. Saving a database in previous formats (<=16.0.0) is disabled.

Application Licensing and Distribution#

If you are interested in Licensing and Distributing your applications, read more about the Application Licensing and Distribution mechanism that BETA CAE Systems has introduced.
